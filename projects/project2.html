The Problem

Astronomical objects not within in our solar system such as exoplanets, comets, asteroids, galaxies, etc are extremely far away, so it is naturally difficult to try and classify them without substantial anaylsis and research. With that being said, I would like to try and predict if a given astronomical object is an exoplanet. I think this dataset archive contains lots of useful information for training decision based machine learning models for that purpose.

Introducing the Dataset

This dataset comes from a larger collection of data that is maintained by NASA Exoplanet Archive. It is a collection of information on known existing exoplanets including name, star host, galaxy, discovery method, discovery year, discovery telescope, and more. It is also a very large dataset with 34951 rows 287 columns, so there is more than enough data available. However, it does contain lots of null values so it will have to go through a cleaning process before it can be used to train a model.
